# Support

For questions and community discussion, use:

- https://github.com/Trace-DB/tracedb/discussions

For bugs or feature requests, open an issue in the repo where the change
belongs. The public repositories cover the downloadable database distribution,
protocol, SDKs, examples, and RFCs.

Hosted TraceDB service operations, gateway code, operator console code, and
production deployment details are not supported through public repositories.
