# TraceDB

TraceDB is an AI-native transactional candidate-stream database.

## Public Distribution

- [`tracedb`](https://github.com/Trace-DB/tracedb): source-available core database distribution.
- [`tracedb-protocol`](https://github.com/Trace-DB/tracedb-protocol): HTTP API, OpenAPI, platform contract, and conformance artifacts.
- [`tracedb-js`](https://github.com/Trace-DB/tracedb-js): official TypeScript SDK.
- [`tracedb-python`](https://github.com/Trace-DB/tracedb-python): official Python SDK.
- [`tracedb-rust`](https://github.com/Trace-DB/tracedb-rust): official Rust SDK.
- [`tracedb-examples`](https://github.com/Trace-DB/tracedb-examples): verified examples.
- [`tracedb-rfcs`](https://github.com/Trace-DB/tracedb-rfcs): public RFCs and design proposals.

## Boundaries

The downloadable TraceDB distribution is public. The hosted TraceDB service,
gateway, operator console, deployment automation, and production operations
code are proprietary service infrastructure and are not part of the public
distribution.

The `tracedb` core repository is source-available under FSL-1.1-ALv2 with a
future Apache-2.0 grant. Protocol, examples, RFCs, and SDK repositories use
Apache-2.0.
